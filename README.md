## 🛠️ Technology Stack

- **Core Runtime**: React (v18) & ReactDOM loaded via CDN for clean component-driven rendering.
- **Styling System**: Vanilla CSS3 for premium visual components + Tailwind CSS CDN utilities for micro-spacings.
- **Helpers**: jQuery for scroll state triggers, smooth animations, and active state navigation.
- **Templating**: HTML5 Semantic markup & dynamic data-driven rendering via local JSON states.

---

## ✨ Key Interactive Features

1. **3D Mouse Tilt Parallax**: Interactive hover perspective tilting on the Hero visual card.
2. **Camera REC Custom Cursor**: Viewport mouse-move bracket tracking with viewport focus states.
3. **Netflix Video Rails**: Uniform vertical video carousels supporting fluid swipe movements.
4. **Gmail Redirection**: Prefilled compose redirection for professional inquiry submissions.

---

## 🚀 Running Locally

Serve the workspace directory with any local static HTTP file server:

```bash
# Using Python
python -m http.server 8080

# Using Node.js (npx)
npx http-server -p 8080
```

Access the application in your browser at `http://localhost:8080`.
